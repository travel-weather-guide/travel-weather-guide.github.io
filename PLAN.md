# Travel Weather - 구현 플랜

> 큰 틀만 정의. 구체적 구현 체크리스트는 PLAN-CURRENT.md에서 관리.
> 각 항목은 2~5줄로 목적과 범위를 알 수 있게 작성.
> CLAUDE.md가 프로젝트의 단일 진실 소스.

---

## Phase 1: 기반 구축 (MVP) (진행 중)

- [ ] **1.1 프로젝트 세팅**
  Next.js 15 + Tailwind v4 + TypeScript strict 초기화.
  디렉토리 구조(components, data, types, utils), 한글 폰트(Pretendard/Noto Sans KR), 글로벌 CSS 변수(하늘색/코랄 팔레트) 설정.

- [ ] **1.2 타입 정의**
  CLAUDE.md 데이터 구조 초안 기반으로 핵심 인터페이스 작성.
  Country, Region, MonthlyData, TravelComment, MonthlyRecommendation, Continent 타입.

- [ ] **1.3 데이터 파이프라인**
  Open-Meteo / REST Countries API에서 기후·국가 데이터를 수집하는 스크립트 작성.
  먼저 5개국(일본, 태국, 프랑스, 미국, 호주)으로 파이프라인 검증 후 전체 확장.
  여행 코멘트(적합도, 팁, 주의사항)도 스크립트로 자동 생성.

- [ ] **1.4 레이아웃**
  Header(로고, 네비게이션), Footer(데이터 출처 크레딧) 공통 컴포넌트.
  모바일 햄버거 메뉴 포함 반응형 네비게이션.

- [ ] **1.5 홈페이지**
  히어로 섹션 + 월 선택 바(1~12월) + 해당 월 추천 목적지 카드 그리드.
  MonthSelector, DestinationCard 공통 컴포넌트 + 데이터 로드 유틸리티.

- [ ] **1.6 국가 목록**
  대륙별 필터 탭으로 국가 카드 그리드 표시.
  국가 클릭 시 상세 페이지 이동.

- [ ] **1.7 국가 상세**
  국가 기본정보(수도, 통화, 비자 등) + 월별 날씨 테이블 + 베스트 시즌 배지.
  하위 지역 목록 + 월별 여행 코멘트 타임라인. generateStaticParams로 정적 빌드.

- [ ] **1.8 지역 상세**
  12개월 날씨 데이터 상세 + 월별 여행 적합도(1~5점) 시각화.
  팁, 주의사항, 이벤트, 옷차림 등 여행 코멘트 전체 표시. generateStaticParams로 정적 빌드.

---

## Phase 2: 핵심 기능 완성

- [ ] **2.1 날씨 차트**
  차트 라이브러리 선정(Recharts vs Chart.js) 후 기온 라인 차트 + 강수량 바 차트 구현.
  국가/지역 상세 페이지에 차트 추가.

- [ ] **2.2 세계지도**
  Leaflet + react-leaflet + OpenFreeMap 타일로 인터랙티브 지도.
  국가 클릭 시 상세 이동, 월 선택 시 추천도 색상 표시(초록~빨강). dynamic() SSR 비활성화 필수.

- [ ] **2.3 월별 추천 페이지**
  `/month/[month]` 경로. 베스트 목적지 TOP 10 + 카테고리별 추천.
  숨은 보석 섹션 + 피해야 할 곳(이유 포함). 1~12월 정적 빌드.

- [ ] **2.4 검색**
  Header 검색바에서 국가/지역명 한국어+영어 자동완성.
  정적 데이터 기반 클라이언트 사이드 검색, 디바운스 적용.

- [ ] **2.5 목적지 비교**
  최대 3개 목적지 선택 후 월별 데이터 나란히 비교 테이블 + 차트 오버레이.
  URL 쿼리로 비교 상태 공유 가능.

- [ ] **2.6 코멘트 강화**
  Meteocons 아이콘 매핑, 축제/이벤트 배지, 옷차림 시각화.
  관광객 밀집도·물가 수준 인디케이터.

- [ ] **3.1 테마별 추천**
