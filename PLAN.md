## Phase 2: 핵심 기능 완성

- [ ] **2.1 날씨 차트**
  차트 라이브러리 선정(Recharts vs Chart.js) 후 기온 라인 차트 + 강수량 바 차트 구현.
  국가/지역 상세 페이지에 차트 추가.

- [ ] **2.2 세계지도**
  Leaflet + react-leaflet + OpenFreeMap 타일로 인터랙티브 지도.
  국가 클릭 시 상세 이동, 월 선택 시 추천도 색상 표시(초록~빨강). dynamic() SSR 비활성화 필수.

- [ ] **2.3 월별 추천 페이지**
  `/month/[month]` 경로. 베스트 목적지 TOP 10 + 카테고리별 추천.
  숨은 보석 섹션 + 피해야 할 곳(이유 포함). 1~12월 정적 빌드.

- [ ] **2.4 검색**
  Header 검색바에서 국가/지역명 한국어+영어 자동완성.
  정적 데이터 기반 클라이언트 사이드 검색, 디바운스 적용.

- [ ] **2.5 목적지 비교**
  최대 3개 목적지 선택 후 월별 데이터 나란히 비교 테이블 + 차트 오버레이.
  URL 쿼리로 비교 상태 공유 가능.

- [ ] **2.6 코멘트 강화**
  Meteocons 아이콘 매핑, 축제/이벤트 배지, 옷차림 시각화.
  관광객 밀집도·물가 수준 인디케이터.

## Phase 3: 고도화

- [ ] **3.1 테마별 추천**

- [ ] **3.4 다국어**
  한국어/영어 i18n. 라우팅 분리(`/ko/...`, `/en/...`) + 언어 전환 UI.

- [ ] **4.2 베스트 시즌 캘린더**
  12개월 캘린더 뷰에서 각 목적지 베스트 시즌 색상 표시.

- [ ] **4.3 짐 싸기 체크리스트**

- [ ] **4.4 배포**
  Cloudflare Pages 설정 + @cloudflare/next-on-pages + 빌드/배포 테스트.

- [ ] **4.5 PWA**
  서비스 워커 + 매니페스트. 정적 데이터 오프라인 캐시.

  홈페이지 또는 별도 페이지에 가격순/추천순 정렬 UI.
